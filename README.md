# TO-DO List with Calendar (Flask)

## 프로젝트 소개  
Flask로 만든 심플한 캘린더 기반 할 일 관리 웹 애플리케이션입니다.  
월별 캘린더와 할 일 목록을 함께 확인할 수 있고, 할 일 추가, 수정, 삭제, 완료 표시 기능을 제공합니다.

## 주요 기능  
- 월별 캘린더 표시 (일요일 시작)  
- 날짜별 할 일 표시  
- 할 일 추가, 수정, 삭제  
- 할 일 완료 체크 및 표시  
- 이전/다음 달로 이동  

## 설치 및 실행 방법  
1. Python 3 설치  
2. Flask 설치:  
```bash
pip install flask

Flask Calendar To-Do List
프로젝트 소개
